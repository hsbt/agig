# Agig

Agig is another Github IRC Gateway, forked cho45's [gig.rb](https://github.com/cho45/net-irc/blob/master/examples/gig.rb)

## Installation

Add this line to your application's Gemfile:

    gem 'agig'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install agig

## Usage

    $ agig -d
